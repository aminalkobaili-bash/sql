## 1. LEFT и RIGHT JOIN

- Получить список пиццерий с рейтингом, которые отсутствуют в записях посещений.
- Использовать LEFT или RIGHT JOIN.
- Ограничения:
  - не применять `IN`, `EXISTS`, `UNION`, `INTERSECT`, `EXCEPT`.

---

## 2. Поиск пропущенных данных

- Найти отсутствующие даты посещений в диапазоне:

```bash
2022-01-01 — 2022-01-10
```

- Условия:
  - учитывать пользователей с `id = 1` и `id = 2`
- Использовать:

```bash
generate_series(...)
```

- Отсортировать по дате.

---

## 3. FULL JOIN

- Получить объединённые данные:
  - пользователи (включая отсутствующие записи)
  - пиццерии (включая отсутствующие записи)
- Период:

```bash
2022-01-01 — 2022-01-03
```

- Заменить `NULL` значения на:

```bash
'-'
```

- Добавить сортировку по всем колонкам.

---

## 4. Использование CTE

- Переписать запрос из раздела 2 с использованием:

```bash
WITH
```

- Генерация дат выполняется внутри CTE.
- Результат должен совпадать с предыдущей реализацией.

## 5. Фильтрация по значениям

- Получить список:
  - названия пицц
  - пиццерии
  - цены
- Условия:
  - пиццы с грибами или пепперони
- Сортировка:
  - по названию пиццы
  - по названию пиццерии

## 6. Анализ данных пользователей

- Выбрать имена:
  - женщины
  - возраст > 25
- Отсортировать по имени.

---

## 7. Фильтрация по нескольким условиям

- Получить список:
  - названия пицц
  - названия пиццерий
- Условия:
  - заказы пользователей: Denis или Anna
- Сортировка по двум колонкам.

---

## 8. Условия по цене и дате

- Определить пиццерию:
  - посещение пользователем Dmitriy
  - дата: `2022-01-08`
  - цена пиццы < 800

---

## 9. Комбинированные условия

- Найти пользователей:
  - мужчины
  - город: Москва или Самара
  - заказывали:
    - pepperoni
    - mushroom
    - или оба варианта
- Сортировка:
  - по имени в обратном порядке.

---

## 10. Пересечение интересов

- Найти пользователей:
  - женщины
  - заказывали одновременно:
    - cheese pizza
    - pepperoni pizza
- Сортировка по имени.

---

## 11. Поиск совпадений по адресу

- Найти пары пользователей:
  - проживающих по одному адресу
- Вывести:
  - имя первого пользователя
  - имя второго пользователя
  - общий адрес
- Сортировка по всем колонкам.
