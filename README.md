# SQL Database Practice (PostgreSQL)

This repository contains a collection of SQL projects focused on database design, querying, optimization, and transaction management using PostgreSQL.

## Overview

The projects cover a wide range of database topics, начиная с базовых SELECT-запросов и заканчивая продвинутыми аспектами, такими как индексация, транзакции и анализ производительности.

## Topics Covered

### Basic SQL

- SELECT queries
- filtering and sorting
- subqueries
- CASE expressions

### Joins and Sets

- INNER / LEFT / RIGHT / FULL JOIN
- UNION / INTERSECT / EXCEPT
- working with duplicates

### Data Manipulation (DML)

- INSERT
- UPDATE
- DELETE
- bulk operations

### Advanced SQL

- Common Table Expressions (CTE)
- window functions
- recursive queries

### Indexing and Optimization

- B-Tree indexes
- functional indexes
- multi-column indexes
- query performance analysis (EXPLAIN ANALYZE)

### Analytics

- aggregation functions
- grouping and filtering
- calculated metrics

### Transactions and Concurrency

- transaction control (BEGIN, COMMIT)
- isolation levels
- concurrency issues:
  - lost update
  - dirty reads
  - non-repeatable reads
  - phantom reads
- deadlocks

## Tools Used

- PostgreSQL
- psql
- SQL

## Skills Demonstrated

- writing efficient SQL queries
- database design understanding
- performance optimization
- data analysis
- working with concurrent transactions
