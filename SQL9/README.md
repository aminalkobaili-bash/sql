## 1. Подсчёт посещений

- Определить:

  - `person_id`
  - количество посещений

- Использовать агрегатную функцию подсчёта.
- Выполнить сортировку:
  - по количеству посещений (убывание)
  - по `person_id` (возрастание)

---

## 2. Вывод имён пользователей

- Модифицировать предыдущий запрос:
  - заменить `person_id` на имя пользователя
- Ограничить результат:
  - вывести топ-4 по количеству посещений
- Выполнить сортировку по имени.

---

## 3. Статистика по пиццериям

- Получить:

  - топ-3 пиццерии по количеству заказов
  - топ-3 пиццерии по количеству посещений

- Объединить результаты в один набор.
- Добавить поле:

```bash
action_type
```

- Значения:

  - `order`
  - `visit`

- Сортировка:
  - по типу действия
  - по количеству (убывание)

---

## 4. Объединённая статистика

- Рассчитать:

  - общее количество заказов и посещений для каждой пиццерии

- Учесть:

  - возможное отсутствие данных в одной из таблиц

- Сортировка:
  - по общему количеству (убывание)
  - по названию (возрастание)

---

## 5. Фильтрация агрегированных данных

- Вывести:

  - имя пользователя
  - количество посещений

- Условие:

  - значение > 3

- Ограничение:
  - не использовать `WHERE`

---

## 6. Уникальные пользователи

- Получить список:

  - уникальных имён пользователей, совершивших заказы

- Ограничения:

  - без использования:
    - `GROUP BY`
    - операций над множествами

- Сортировка:
  - по имени

---

## 7. Метрики пиццерий

- Для каждой пиццерии определить:

  - количество заказов
  - среднюю цену
  - максимальную цену
  - минимальную цену

- Округлить среднюю цену до двух знаков.
- Сортировка:
  - по названию

---

## 8. Средний глобальный рейтинг

- Рассчитать:

  - среднее значение рейтинга по всем пиццериям

- Название поля:

```bash
global_rating
```

- Округление:
  - до 4 знаков после запятой

---

## 9. Анализ заказов по адресам

- Получить:

  - адрес пользователя
  - название пиццерии
  - количество заказов

- Предположение:

  - пользователь посещает пиццерии своего города

- Сортировка:
  - по адресу
  - по названию пиццерии

---

## 10. Вычисляемые показатели

- Для каждого адреса определить:
  - вычисляемое значение:

```bash
max(age) - (min(age) / max(age))
```

- средний возраст
- результат сравнения:

  - формула > среднее значение

- Сортировка:
  - по адресу
