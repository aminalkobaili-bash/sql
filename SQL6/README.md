## 1. Моделирование графа

- Создать таблицу для хранения структуры графа:
  - начальная вершина (`point1`)
  - конечная вершина (`point2`)
  - стоимость перехода (`cost`)
- Добавить данные:
  - для всех направлений между узлами
  - стоимость должна быть одинаковой в обе стороны

---

## 2. Поиск минимального маршрута

- Реализовать поиск маршрутов:

  - старт из вершины `a`
  - посещение всех узлов ровно один раз
  - возврат в начальную точку

- Использовать:

```bash
WITH RECURSIVE
```

- Требования к результату:

  - вывести:
    - суммарную стоимость (`total_cost`)
    - маршрут (`tour`)
  - маршрут должен включать полный цикл (например: `a -> b -> c -> d -> a`)

- Определить:

  - маршруты с минимальной суммарной стоимостью

- Добавить сортировку:
  - сначала по `total_cost`
  - затем по маршруту

---

## 3. Анализ маршрутов с максимальной стоимостью

- Модифицировать предыдущий запрос:

  - дополнительно вывести маршруты с максимальной стоимостью

- Требования:

  - сохранить минимальные маршруты
  - добавить маршруты с наибольшей стоимостью
  - итоговый результат объединяет оба набора

- Добавить сортировку:
  - по `total_cost`
  - по маршруту
